运行方式：

       切换目录到脚本目录，或使用绝对路径  运行以下命令
-----------------------------------------------------------------------------------------------
           python report.py InputPath [OutputPath]
-----------------------------------------------------------------------------------------------
    e.g:  python   ./report.py  ~/user/data   ~/user/report

========================================================

        1、参数之间用空格
        2、OupputPath默认为当前路径
        3、如果目录不为空，则会创建report目录并输出
                3.1  序列  > 300bp 的 文件名为  report_over_300.txt
	3.2  每个FASTA文件中, 最长最短的序列输出到   longest_shortest.fasta
	3.3  GC% 和 N50 报告   GC_and_N50_report.txt
